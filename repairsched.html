Repair Check-in











How to Use:

  • "Check-in a new repair"- Adds a repair to the repair queue for the current available slot.
  • "Change Maximum Repairs"- Changes the amount of work techs can accomplish per slot.
  • "Next Slot >"- If a time slot goes unfulfilled or a repair is checked in at the end of this slot period, this button will start a new cycle in the next slot.
